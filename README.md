# XyharaDB

XyharaDB adalah library database custom dengan query bahasa Indonesia yang mudah dipahami.  
Mendukung operasi: TAMBAH KE, LIHAT SEMUA DARI, UBAH … SET … DIMANA …, HAPUS DARI … DIMANA …

## Cara pakai
1. Jalankan console: `node console.js`
2. Contoh query:
TAMBAH KE pengguna (id, nama, umur) NILAI (1, 'KHara', 16) LIHAT SEMUA DARI pengguna UBAH pengguna SET nama = 'Xyhara Studio' DIMANA id = 1 HAPUS DARI pengguna DIMANA id = 1
